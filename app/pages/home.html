title: Welcome to Flattery ---

Flattery is ready to go!

Start by adding pages to the pages directory! (This page is pages/home.html)

The menu at the top here is the primaryNavigation menu. Edit it in app/data/config/navigation.yml

Adding users

To add a user, open the directory of flattery in your terminal and run php flattery.php users:create. Alternatively, add the user in app/data/users.yml.

Custom routes

If you need a page to respond to POST requests for handling form requests, consider defining a route in app/routes.php. It works similarly to Laravel - if that's greek to you, it's quite simple:

Defining a custom route:

  1. Open app/routes.php
  2. Here, use router() to get Flattery's ThowsenMedia/Flattery/HTTP/Router instance.
  3. Define routes by calling GET or POST to the router, and giving it a URI and a callback function.
  4. For example: router()->get('/hello', function() {return 'world!';});

Theming

For more advanced usage, you can copy this theme (themes/strappery) to make your own:

  1. Copy themes/strappery to themes/mytheme
  2. Rename themes/mytheme and
    1. Rename strappery.php to mytheme.php, and the strappery.yml to mytheme.yml

Blocks

Blocks are sections of a theme template whose content can be defined either in data/themes/[themename].yml (in the blocks section) or by defining the block as a template file in app/blocks/[block_name].php if dynamic content is needed.

To see which blocks are available in a given theme, open the theme's YML file in themes/[theme]/[theme].yml.

Plugins

Plugins are stored in /plugins. They consist of a single class file and a yml file. Open up the /plugins and take a look!
